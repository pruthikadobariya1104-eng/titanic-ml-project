# Titanic Survival Prediction Using Machine Learning

## Project Overview
This project uses Machine Learning algorithms to predict whether a passenger survived the Titanic disaster based on passenger details.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Algorithms
- Random Forest Classifier
- Decision Tree Classifier

## Features
- Data preprocessing and cleaning
- Model training and testing
- Accuracy evaluation
- Confusion Matrix visualization
- Feature Importance graph
- Actual vs Predicted comparison
- ROC Curve analysis

## Dataset
Titanic dataset containing passenger information such as:
- Passenger Class
- Gender
- Age
- Fare
- Siblings/Spouses
- Parents/Children

## Output
The model predicts passenger survival and generates visualization graphs for performance analysis.

## Project Structure
- Titanic_Prediction.ipynb
- titanic_cleaned.csv
- predictions.csv
- dashboard.png
- roc_curve.png
